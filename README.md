# KOSPI 200 파생상품 트래커 — 공개 데이터

이 저장소는 [KOSPI 200 파생상품 트래커](https://kospi200-tracker.pplx.app) 대시보드가 사용하는 **공개 데이터 파일**만 보관합니다.

- `dashboard.json` — 집계 결과(누적 포지션, 투자자별 수급, 방향성 신호). 평일 장 마감·정산 데이터를 다음 영업일 오전 08:00 KST에 자동 갱신합니다.
- 지수선물 데이터는 3·6·9·12월 분기 만기 주기, 지수옵션 데이터는 매월 둘째 목요일 만기 주기 안에서만 집계됩니다.

원본 소스 코드와 파이프라인은 별도 비공개 저장소에서 관리됩니다. 이 저장소에는 인증 정보/API 키/원시 개인 데이터가 포함되지 않습니다.
